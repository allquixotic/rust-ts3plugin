The trait that has to be implemented by a plugin. To enhance a library to a working TeamSpeak plugin you have to call the macro create_plugin! afterwards.

fn captured_voice_data(&mut self, api: &mut TsApi, server_id: ServerId, samples: &mut [i16], channels: i32, send: &mut bool) -> bool

The recorded sound from the current capture device. send is set if the audio data will be send to the server. This attribute can be changed in this callback. The return value of this function describes if the sound data was altered. Return true if the sound was changed and false otherwise. The callbacks with audio data are called from another thread than the other functions.
